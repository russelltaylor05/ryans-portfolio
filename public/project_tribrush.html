Tri-Brush Scrap Removal System

Design, Prototype, Full Scale Production of Adujstable Scrap Removal System

Description

This project involved the design, prototype, and scaling to full production of a product known as the “Tri-Brush” system. This is a manually operated pivoting brush set for the company’s Classic Layboy machines that can be safely adjusted during full production in order to separate scrap from the product flow. The guidelines for this project were to design a product that was to be standard on all new machines as well as retrofitable to older vintage machines. The design was prototyped, refined, and implemented for full scale production.

Project Highlights

  • Locking Handle
  • Tri-Tilt Linkage
  • No Sag Tensioner
